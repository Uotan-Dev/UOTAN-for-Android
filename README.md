# UOTAN for Android

UOTAN for Android (柚坛社区安卓端) 是一款基于 Kotlin, Android View 和 Jsoup 的安卓应用

通过请求、爬取、整理三个步骤让您在手机上更舒适的访问柚坛社区

## 涉及到的 使用/修改/灵感 的开源项目

**[AndroidX](https://github.com/androidx/androidx)**
